How to Play:

Click Create Game to generate a room code, and it will auto-fill.
Otherwise, enter the code of the room that the host made. Then insert your name and click Join Game.
If you created the game and everyone is ready to start, click Everyone Is Ready. The goal of the game is to find the Afikoman, wrapped in a green napkin. When you find it, click on it and you win!
Want to play again? As host, click Play Again to have everyone play again. If a player doesn't want to play agin, they can click Return to Home.
There's 3 maps with four locations each, so you might get a repeat.
We recommend that you don't play on mobile devices, but if you do, make sure to be in horizontal mode!
