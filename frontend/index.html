How to Play:

Click Create Game to generate a room code. It will auto-fill if you created it. Otherwise, enter the code of the room that the host made. Then insert your name and click Join Game. If you created the game and everyone is ready to start, click Start. The goal of the game is to find the Afikoman, wrapped in a green napkin. When you find it, click on it and you win!
There's 3 maps with four locations each, so you might get a repeat.
Playing on mobile? Make sure to be in horizontal mode!
